## Methodology

The assessment followed a structured consulting-style methodology:

1. External Reconnaissance  
2. Attack Surface Mapping  
3. Passive Vulnerability Identification  
4. Security Header Evaluation  
5. Configuration Review  
6. Risk Classification  
7. Remediation Planning  

The goal was to identify weaknesses that could increase organizational risk while maintaining strict ethical boundaries.
